Ciberseguridad 02

Ataques de Ciberseguridad

Ataque Descripción Impacto Ejemplo
📧
Phishing
Alto Riesgo
 Engaña al usuario para robar las credenciales mediante correos falsos. Robo de identidad, pérdida de datos. Correo falso de un banco solicitando datos.
🦠
Malware
Crítico
 Software malicioso que daña o toma el control del sistema. Pérdida de datos, control del sistema. Un troyano oculto en un archivo adjunto.
🔒
Ransomware
Crítico
 Secuestra datos y exige pago para liberarlos. Secuestro de datos, pérdidas económicas. WannaCry afectó a miles de sistemas en 2017.
⚡
Denegación de servicio (DoS)
Alto Riesgo
 Inunda un servidor con tráfico para hacerlo inaccesible. Interrupción de servicios. Ataque a un sitio web de gobierno.
💥
DDoS
Crítico
 Ataque DoS desde múltiples dispositivos (botnet). Caída masiva de servicios. Ataque a Dyn que afectó a Twitter y Netflix.
💉
Inyección SQL
Alto Riesgo
 Inserta código SQL malicioso para acceder a bases de datos. Acceso a datos confidenciales. Obtener contraseñas desde un formulario vulnerable.
🔨
Fuerza bruta
Medio Riesgo
 Adivina contraseñas por prueba y error. Acceso no autorizado. Script probando miles de contraseñas.
🕵️
Man-in-the-Middle (MitM)
Alto Riesgo
 Intercepta la comunicación entre dos partes. Robo o alteración de información. Capturar datos en una red Wi-Fi pública.
🎯
Zero-Day
Crítico
 Aprovecha vulnerabilidades aún no conocidas. Compromiso del sistema sin solución inmediata. Ataque antes de que salga un parche de seguridad.
⌨️
Keylogging
Alto Riesgo
 Registra pulsaciones del teclado. Robo de contraseñas y datos personales. Software oculto que graba todo lo que escribes.
💻
XSS (Cross-Site Scripting)
Medio Riesgo
 Inyecta scripts maliciosos en páginas web. Robo de cookies o redirecciones maliciosas. Script en un comentario que roba la sesión.
👥
Ingeniería social
Alto Riesgo
 Manipula a personas para obtener acceso o datos. Filtración de información, acceso físico. Alguien se hace pasar por técnico de soporte.
